# Clinical Critical Calculator - GitHub Pages Deployment

## Live Site
**URL**: https://astrobsm.github.io/CRITICALCARECALCULATOR/

## Deployment Details
- **Platform**: GitHub Pages
- **Branch**: main
- **Build**: Automatic via GitHub Actions
- **Domain**: astrobsm.github.io/CRITICALCARECALCULATOR
- **HTTPS**: Enforced

## Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Test production build locally
npm run start
```

## GitHub Pages Configuration
1. Source: Deploy from a branch
2. Branch: main / root
3. Custom domain: Not configured (using default)
4. HTTPS: Enforced

## Automatic Deployment
Every push to the `main` branch triggers an automatic build and deployment via GitHub Actions.

## Manual Deployment
To manually trigger a deployment:
1. Go to GitHub repository
2. Click "Actions" tab
3. Select "Deploy to GitHub Pages" workflow
4. Click "Run workflow"
